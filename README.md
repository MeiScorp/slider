
# 📸 Modern Image Slider (HTML, CSS & JavaScript)

A clean, responsive, and fully custom image slider built with pure HTML, CSS, and JavaScript.  
The project demonstrates DOM manipulation, event handling, dynamic slide switching, and modern UI styling with smooth transitions and interactive controls.

This slider is lightweight, easy to customize, and suitable for landing pages, portfolios, galleries, or any modern website.

you can run it here:
https://meiscorp.github.io/slider/
---

## ✨ Features

### ▶️ Smooth Slide Switching  
- Next/previous navigation  
- Clickable dot indicators  
- Active slide highlighting  

### 🎨 Modern UI & Animations  
- Glassmorphism-inspired arrow buttons  
- Smooth fade transitions  
- Rounded corners and soft shadows  
- Clean, minimalistic layout  

### 📱 Fully Responsive  
- Scales smoothly on desktop, tablet, and mobile  
- Images adapt using `object-fit: cover`  

### 🧩 Pure JavaScript Logic  
- No external libraries  
- Simple and readable code  
- Easy to extend with autoplay, swipe gestures, or captions
